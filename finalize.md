## 8. Deploy Guide

Проект — vite-static. Рекомендованные хосты:

### Netlify (рекомендован)
```
Build command:  npm run build
Publish dir:    dist
```
Перетащить папку `dist/` в Netlify Drop — мгновенный деплой без настройки.
Переменные окружения добавить в: Site Settings → Environment Variables.

### Vercel
```bash
npx vercel --prod
# Переменные окружения: vercel env add VITE_FORM_ENDPOINT
```

### GitHub Pages
```bash
npm run build
# Содержимое dist/ → ветка gh-pages
```

### VPS / хостинг
Залить содержимое `dist/` в корень публичной директории (public_html, www и т.д.).
Nginx: добавить `try_files $uri $uri/ /index.html;` для корректной обработки путей.

## 9. Security / Performance Headers (Nginx пример)

```nginx
add_header X-Frame-Options "SAMEORIGIN";
add_header X-Content-Type-Options "nosniff";
add_header Referrer-Policy "strict-origin-when-cross-origin";
add_header Permissions-Policy "camera=(), microphone=(), geolocation=()";

# Кэширование ассетов (имена с хешем из Vite)
location /assets/ {
  expires 1y;
  add_header Cache-Control "public, immutable";
}
```
