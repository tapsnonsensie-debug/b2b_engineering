# project-tz.md — ТехПоток

## 1. Паспорт проекта

- **Компания:** Инжиниринговая группа «ТехПоток»
- **Ниша:** B2B-инжиниринг
- **Город/регион:** Нижний Новгород
- **Описание:** Проектирование, внедрение, автоматизация, сервисное сопровождение для производственных и коммерческих компаний
- **Цель сайта:** Сбор заявок на аудит, консультацию или расчёт проекта
- **Главное действие:** Получить консультацию / запросить аудит

**Целевая аудитория:**
- Руководители производств и технические директора
- Собственники бизнеса — подрядчик по внедрению инженерных решений
- Закупщики, сравнивающие подрядчиков по компетенциям и кейсам

## 3. Дизайн-направление

- **Визуальный тон:** Индустриальный high-end, технологичный, сдержанный, структурный. Надёжность, инженерия, B2B-компетентность, контроль процессов
- **Цветовая палитра:** Графит, белый, холодный синий, акцентный бирюзовый или оранжевый
- **Типографика:** Современный гротеск, заголовки плотные, межстрочность 140–160%, масштаб через `clamp()`
- **Логотип:** `[logo]` — сдержанный wordmark или инженерный знак без банальных шестерёнок
- **Фото:** `[16:9 image]` — производственные объекты, инженерные узлы, команда на объекте, технологичное оборудование. Реалистично, без визуального шума
- **Иконки:** Инфографика, диаграммные блоки, пиктограммы отраслей и процессов
- **Запреты:** Без теней-перегруза, градиентов, стекломорфизма

---

## 4. Состав секций (порядок)

1. Header — логотип, навигация, телефон `[phone]`, кнопка «Получить консультацию»
2. Hero — оффер, отрасли, CTA, доверительные маркеры
3. Для кого решение — отрасли и сценарии применения
4. Услуги / компетенции — карточки направлений
5. Кейсы — задачи → что сделали → результат
6. Процесс работы — этапы внедрения, линейная схема 5–6 шагов
7. Преимущества — сроки, стандарты, документация, сервис
8. Команда / эксперты / сертификаты
9. FAQ
10. Контакты + карта (при наличии точки для встреч)
11. Footer — legal-ссылки, реквизиты-заглушки, кнопка «Наверх»

**Desktop-логика:**
- Hero: слева оффер, справа техно-композиция или фото оборудования
- Компетенции — сетка карточек
- Кейсы — строгие вертикальные карточки или слайдер
- Процесс — линейная схема 5–6 шагов
- Контакты — карта + данные компании

**Квиз:** нет — B2B-пользователь ценит скорость и конкретику. Достаточно основной формы + popup

**Спецкомпоненты:**
- Popup консультации
- Слайдер кейсов
- Карточки компетенций с отраслями
- FAQ accordion

**Правила адаптации:**
- Сохранять деловой и чистый вид без перенасыщения схемами
- Инфографика упрощается до карточек с короткими тезисами
- Форма короткая и удобная для быстрого обращения
